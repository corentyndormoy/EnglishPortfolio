Canawish Intranet

Camp Canawish is a camp reserved for physically or mentally handicapped people to spend great days during summer in Quebec (Canada).

Using an application on a very old computer to manage their camp, they were looking for a more modern solution before their computer goes away. This first application was the only way for them to manage the camp: this loss could stop the camp for a while.

In an apprenticeship, VASSELIN Guilhem and I had to produce this solution from the Camp Canawish managers need and their old application. Made in native PHP and some JavaScript, this Intranet permits to save several information to manage the camp such like clients, booking and stays.

2D Game

Highly inspired by Mario Bros, I have created a small 2D game using Unity where the player got to jump over monsters to collect points.

One more monster is spawning each wave. Randomly, a few items such like coins which increases the score, or a heart which increases the lives count, will appear during the game.

The game is over once the time is up or when the player lost all of his life against monsters.
